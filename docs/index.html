重点能力：支持定制编排业务流程

这是 FishAutoPro 最核心的差异化能力。你不是只能选“自动发货开/关”，而是可以把你的真实业务链路拆成节点，由系统按顺序自动执行。

当前可编排的流程节点

已经支持触发、等待回复、发货、提取变量、API 请求、条件判断、延迟、通知、标记发货等节点。适合大多数自动交付业务。

触发 等待买家回复 提取关键信息 API 请求 条件判断 发货 延迟 通知 标记发货

一个典型流程示例

1
买家下单后触发流程，先发送提示语，要求回复手机号或邮箱
2
系统等待买家回复，收到后自动提取关键信息并保存成变量
3
把变量传给外部 API 取货或下发业务，再把返回值拼入发货内容
4
成功则继续发货并标记订单已发货；失败则走兜底分支，提醒人工处理

不是固定模板，而是可按需编排

你可以针对不同商品做不同交付流程，不再被单一发货模式限制。

支持带条件和失败兜底

流程不是一条死路，提取失败、接口失败、条件不满足，都可以分支处理。

适合扩展你的业务系统

如果你有自己的库存系统、选号系统、代充系统、面板接口，都可以对接进来。
