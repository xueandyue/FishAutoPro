# FishAutoPro

FishAutoPro 是面向闲鱼多账号运营的自动化管理平台，主打 **AI 客服自动回复、自动发货、免拼单发货、订单/会话管理、商品管理、流程编排、知识库、自动评价、自动擦亮、自动补货、商品监控和秒杀商品、通知告警、授权管理和多端部署**。

它适合做闲鱼虚拟商品、卡密资源、教程资料、代充业务、账号交付、多店铺客服等场景。系统支持接入 OpenAI 兼容大模型，让 AI 基于商品详情、知识库、历史会话和订单信息回复买家；也支持等待买家回复、提取变量、调用外部 API、条件判断、失败兜底和通知提醒等自动化流程。

## 核心能力

- 闲鱼多账号统一管理：账号状态、商品、订单、会话、在线聊天集中处理。
- AI 客服自动回复：支持商品上下文、知识库、历史会话、订单信息和人工接管边界。
- 自动发货：支持固定文本、库存卡密、接口取货、免拼单发货和复杂交付流程。
- 流程编排：支持等待回复、变量提取、API 请求、条件判断、延迟、通知、标记发货。
- 运营自动化：支持自动评价、一键擦亮、定时擦亮、求小红花、自动补货、商品监控和秒杀商品、退款自动回复和消息通知。
- 轻量部署挂机：支持 Windows、macOS、Linux、OpenWrt、ddwrt、安卓手机和 Docker上本地部署，解决IP关联问题
